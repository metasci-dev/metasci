===
ACE
===

The ACE file format is a continuous-energy cross-section library.  It is used in codes such as MCNP 
and Serpent.  ACE files are typically generated using the ``ACER`` module of Njoy.  This module 
provides functions for inspecting such files.  For more information about the structure of ACE
files, please see Appendix F of the MCNP manual.
